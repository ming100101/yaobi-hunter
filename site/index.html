閱讀風險聲明 →
01

本機優先設定、記錄與策略證據放在 `%LOCALAPPDATA%\YaobiHunter`。

02

訊號有來源TG 卡價不是成交價，確認盤與舊紙盤分開呈現。

03

失敗也保留資料不足、失效、過期與走車不會被靜默刪走。

04

沒有盈利承諾所有歷史與模擬結果都可能無法重現。

三步開始

下載、解壓、開啟。

  1. 1
    下載最新版 ZIP

    只使用官方 GitHub Releases，並對照 SHA-256。

  2. 2
    解壓後開啟 EXE

    App 會在本機瀏覽器視窗打開。首次執行可能見到 SmartScreen。

  3. 3
    自行選擇 Telegram

    通知是可選功能；Bot Token 與 Chat ID 只保存在你的電腦。
